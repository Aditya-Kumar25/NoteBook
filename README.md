# NoteBook

A frontend application built with React for organizing programming notes in a simple and structured way. The goal of this project is to make learning easier by presenting topics chapter-wise with a clean and easy-to-use interface.

## Current Status

The frontend is complete and fully functional.

I'm currently adding the chapter content and improving the overall learning experience. More topics and features will be added over time.

## Features

- Responsive UI
- Chapter-based navigation
- Search functionality
- Clean reading interface
- Reusable React components
- Client-side routing

## Tech Stack

- React
- JavaScript
- React Router
- Axios
- CSS / Tailwind CSS *(update if needed)*

## Running the Project

Clone the repository

```bash
git clone https://github.com/Aditya-Kumar25/NoteBook.git
```

Move to the frontend folder

```bash
cd NoteBook/Frontend
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
